Guide d'utilisation

Introduction

Ce système utilise l'IA pour prédire le risque d'abandon scolaire en analysant divers facteurs.

Facteurs analysés

  • Informations personnelles (âge, sexe, niveau scolaire)
  • Performance académique (notes, absences)
  • Contexte familial (situation, travail des parents)
  • Facteurs environnementaux (distance, accès internet)

Comment utiliser

  1. Remplir le formulaire avec les données de l'étudiant
  2. Cliquer sur "Prédire l'Abandon Scolaire"
  3. Analyser les résultats et les conseils
  4. Exporter le rapport si nécessaire

Fonctionnalités avancées

Simulation de scénarios

Testez différents scénarios en modifiant les valeurs du formulaire pour voir leur impact sur le risque d'abandon.

  • • Sauvegardez des scénarios pour les comparer

Explications SHAP

Visualisez l'importance de chaque facteur dans la prédiction grâce aux graphiques SHAP.

  • • Comprenez quels facteurs influencent le plus la prédiction
  • • Identifiez les leviers d'action prioritaires
  • • Justifiez les décisions d'accompagnement

Historique et statistiques

Suivez l'évolution des prédictions et analysez les tendances sur votre établissement.

  • • Consultez l'historique des prédictions

  • • Exportez les données pour analyse approfondie

Conseils d'utilisation

✅ Bonnes pratiques

  • • Utilisez des données récentes et précises
  • • Consultez régulièrement l'historique
  • • Partagez les rapports avec l'équipe
  • • Testez différents scénarios d'amélioration
  • • Documentez les actions mises en place

⚠️ Points d'attention

  • • Ne basez pas les décisions uniquement sur l'IA
  • • Considérez le contexte individuel
  • • Respectez la confidentialité des données
  • • Validez les prédictions par l'observation
  • • Adaptez les interventions au profil
